# 使用方法:


1、下载安装包，安装

2、打开千牛接待窗口（非千牛控制台）

3、用管理员身份运行程序

4、选择催付的千牛店铺，点击搜索订单

5、勾选要催付的订单，填写消息内容，点击发送消息按钮
